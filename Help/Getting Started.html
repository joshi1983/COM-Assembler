Trying out some code

  1. In the program's main menu, click File -> Open.
  2. Open the Programs folder.
  3. Open the Examples folder. Open the Simple_Programs folder. There are a few examples in there to help you learn assembly.
  4. Select Hello World.asm and open it. The comments explain the code.
  5. From the menu again, click Run -> Assemble and Run. A DOS console window should appear containing the message 'Hello, World!'.
  6. Try editing the code to change what the program does. Look at the other examples and see how they work. Try making your own programs. Test out the use of some DOS interrupts explained on the web. You can learn a lot.
